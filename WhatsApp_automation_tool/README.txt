README

The application reads phone numbers from an excel file and sends the WhatsApp message through WhatsApp web.

First, the application asks for the path to the excel file containing phone numbers.

Then it will ask the message.
Then the path to the attachment you want to send (if any)

Then you have to enter the column number containing phone numbers.

Once you click the send message button, it will open the WhatsApp web and you need to scan the QR code from your WhatsApp.
Once you scan the QR code it will start sending messages.

Note that this application requires the chrome browser.
